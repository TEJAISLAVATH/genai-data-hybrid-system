# GenAI + Data Hybrid System

This project demonstrates how AI can analyze structured data and generate insights.

## Overview

This system:
- Reads CSV data
- Understands data
- Answers analytical questions
- Generates insights using AI

## Tech Stack

- Python
- Ollama
- TinyLlama
- Pandas

## Example Questions

- Which product has highest sales?
- Which region has more sales?
- Give insights from the dataset

## How to Run

1. Install Ollama  
2. Run:
   ollama pull tinyllama  

3. Run:
   python app.py
